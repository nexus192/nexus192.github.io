Проекты

C/C++ разработка

В ходе обучения разработал ряд библиотек, включая реализации String и Matrix, а также шаблонные библиотеки структур данных. В работе применял мультипарадигменный подход: процедурное программирование, ООП (с реализацией абстрактных интерфейсов для обработки разнотипных входных данных), шаблонные метапрограммы. Работал со стандартными библиотеками (STL, algorithm, utility, limits), системными вызовами (stdio.h, fstream) и тестовыми фреймворками (gtest.h, check.h).

Разработал несколько desktop-приложений на Qt: классические игры Snake и Tetris с использованием паттернов проектирования, 3D Viewer с базовой визуализацией. Имею опыт работы с Vulkan API. Реализовал мессенджер с клиент-серверной архитектурой (Nginx, Boost.Asio, Qt GUI, Flask для бэкенда), включая обработку HTTP/HTTPS запросов и проектирование взаимодействия между компонентами системы.

Системное администрирование

Настраивал виртуальные среды: развертывание PostgreSQL в Docker, организация сети между VM. Понимаю принципы работы сетевых протоколов (TCP/IP стек, DHCP, SSH, VPN), проектировал систему с учетом требований к сетевой безопасности.

Python разработка

Создал веб-приложение "Крестики-нолики" на Flask с акцентом на масштабируемость и отказоустойчивость. Реализовал асинхронную обработку обработку ввода/вывода, многопоточный режим для тяжелых вычислений.
